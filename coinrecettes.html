Le coin recettes

Le coin recettes est un site fictif pour un site de recettes. À l'aide de React, plusieurs appels sont fait à l'api "themealDB" afin d'avoir accès à plusieurs recettes ainsi que leurs instructions. React Redux permet également de faire une gestion des abonnés.

Technologies utilisées: React, React Redux, Tailwind CSS, CSS

Projet Renova

Description du projet

Accueil

La page d'accueil affiche un carousel installé à l'aide d'un composant de la librairie React-Responsive-Carousel

Les catégories

Cette page permet à l'aide d'un appel à l'API d'afficher toutes les recettes contenues dans celle-ci.

Les recettes selon la catégorie

À partir d'un second appel à l'api, l'url prend en considération le nom de la catégorie offert par l'api pour afficher toutes les catégories offertes par celle-ci.

Instructions de recettes

Cette page permet avec un appel à l'api distinct d'aller chercher l'identifiant de la recette sur laquelle on a cliqué pour afficher l'image de la recette, une liste des ingrédients ainsi que les instructions de la recette.

Abonnement

Cette page permet à l'aide de React Redux de faire un ajout d'abonné. Lorsqu'on entre un abonné, il s'ajoute ensuite dans une carte dans la page "liste des abonnés". Une vérification JavaScript est aussi effectuée pour le formulaire.

Modification de produit

React Redux permet ici de supprimer les cartes d'abonnées sur lesquelles on clique.
