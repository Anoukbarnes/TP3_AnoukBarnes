Pastaio

Pastaio est un site web multipages pour le restaurant Pastaio fait avec HTML5 et CSS3. Le site responsive de plusieurs pages permet au restaurant de présenter son menu, de remplir un formulaire afin de faire une réservation, d'avoir plus d'informations quant au lieu et de préparer une commande.

Projet Pastaio

Description du projet

Page Ma commande

Cette page parmet à l'utilisateur de voir les éléments du menu et faire sa commande. Avec JavaScript, la page permet d'afficher, modifier, supprimer et ajouter de manière dynamique les quantités de sa commande. Cela permet également d'afficher le total de la commande de manière dynamique selon les changements de l'utilisateur.

Page Réservations

La page réservation permet de remplir un formulaire afin de réserver une table. Le formulaire fait avec HMTL5 contient plusieurs types d'entrées et de boutons. Une vérification JavaScript est ensuite complété avec des messages d'erreurs spécifiques pour chaque champ.

Page Restaurant

La page Restaurant affiche une grille d'image fait avec la propriété css grid ainsi que l'utilisation d'un iframe pour afficher la localisation du restaurant sur Google Maps.
