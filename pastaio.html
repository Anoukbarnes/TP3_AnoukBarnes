Pastaio

Pastaio est un site web multipages pour le restaurant fictif Pastaio fait avec HTML5, CSS3 et JavaScript. Le site responsive de plusieurs pages permet au restaurant de présenter son menu, de remplir un formulaire afin de faire une réservation, d'avoir plus d'informations quant au lieu et de préparer une commande.

Projet Pastaio

Description du projet

Page Ma commande

Avec JavaScript, la page permet d'afficher, modifier, supprimer et ajouter de manière dynamique les quantités de la commande du client tout en affichant le total de celle-ci de manière dynamique selon les changements de l'utilisateur.

Page Réservations

La page réservation permet de remplir un formulaire afin de réserver une table. Une vérification JavaScript est ensuite complété avec des messages d'erreurs spécifiques pour chaque champ.

Page Restaurant

La page Restaurant affiche une grille d'image fait avec la propriété css grid ainsi que l'utilisation d'un iframe pour afficher la localisation du restaurant sur Google Maps.
